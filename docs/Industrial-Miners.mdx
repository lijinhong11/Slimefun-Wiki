# 工业矿机 {#industrial-miners}

工业矿机（Industrial Miner）是 Slimefun 的[基础机器](/Basic-Machines)之一。

它会从上往下寻找并挖掘附近的矿石，然后将它们放到箱子里。

你需要在箱子内先放置燃料才能让工业矿机运行，可以在 Slimefun 指南中找到可添加的燃料列表。
