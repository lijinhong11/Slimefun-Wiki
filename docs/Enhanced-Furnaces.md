# 强化熔炉 {#enhanced-furnaces}

强化熔炉是 Slimefun 的[基础机器](/Basic-Machines)之一。

它们与原版熔炉功能相同，但有更快的燃烧速度、燃料效率，与更多的产物。

## 强化熔炉等级 {#enhanced-furnace-tiers}

强化熔炉共有1 3 个等级，下方表格详细列出了它们的区别。

| 等级 | 燃烧速度 | 燃料效率 | 产物 |
| ---- | ---------------- | --------------- | ---- |
| I | 1x | 1x | 1x |
| II | 2x | 1x | 1x |
| III | 2x | 2x | 1x |
| IV | 3x | 2x | 1x |
| V | 3x | 2x | 2x |
| VI | 3x | 3x | 2x |
| VII | 4x | 3x | 2x |
| VIII | 4x | 4x | 2x |

| X | 5x | 5x | 2x |
| XI | 5x | 5x | 3x |
