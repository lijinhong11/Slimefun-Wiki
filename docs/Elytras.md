# 鞘翅 {#elytras}

该页面包含以下物品的信息:

- 鞘翅鳞片
- 鞘翅
- 鞘翅（经验修补）
- 鞘翅（灵魂绑定）

## 获取与使用 {#obtaining--usage}

这些物品都可以在[魔法道具](/Magical-Gadgets)分类中找到。你可以在游戏内查看具体的配方。

鞘翅鳞片用于合成，而其他都是可以使用的鞘翅。

| 物品 | 特殊效果     |
| ---- |----------|
| 鞘翅 | 无        |
| 鞘翅（经验修补）| 附带经验修补附魔 |
| 鞘翅（灵魂绑定）| 死亡时不会掉落  |

:::info 有趣的事实

因为大型服务器中鞘翅消耗得很快，所以我们添加了这些物品。新玩家将会很难获得这些物品。

:::
