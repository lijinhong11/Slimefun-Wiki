# 自动点火机 {#automatic-ignition-chamber}

自动点火机（Automatic Ignition Chamber）是 Slimefun 的[基础机器](/Basic-Machines)之一。

## 使用方法 {#usage}

自动点火机只能作为[冶炼炉](/Smeltery)的扩展。它**不能**用于简易冶炼炉。

你只需要将此机器放在冶炼炉的**发射器旁**，然后填充打火石，它就会自动给冶炼炉点火。
